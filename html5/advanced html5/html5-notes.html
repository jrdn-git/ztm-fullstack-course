# ztm/webdev/html5 # key syntax: - - - - - - <h1>-</h6> - <a> # .html formatting - by default, servers will first send index.html file to the browser - good practice to name main .html file 'index.html' ## advanced HTML forms - to create a form use: <form>_</form> - to create a text input field, use: <input type="text"> - to create a submit button, use <input type="submit"> //use "value='Register'" to give the submit value a different text - to reset the fields, use <input type="reset"> // use "value='Clear'" to give the reset value a different text - to make a field required, use 'required' at end of tag <input type="text" required> // this makes user fill out required field/cannot move forward without - in order to ensure email is submitted in the email field, use <input type="email"> - in order to create a date/birthday field, use <input type="date"> - in order to create a gender field, use the radio option <input type="radio"> // to group radio buttons together, use the name attribute <input type="radio" name="gender"> // makes it so that you can click away from other radio buttons to another - in order to create a checkbox in html, use the 'checkbox' attribute <input type="checkbox"> - in order to create a selection list, use <select> <option value="volvo">Volvo</option> </select> // creates a dropdown list to select from // can use 'multiple' in 'select' tag in order to select multiple - in order to create a field for passwords, use <input type="password"> // to give a minimum required length for the password, use: <input type="password" minlength="5"> - form methods (!!research ) -GET (default) -POST
